# Tiffin Tracker

A simple, premium web application to track your tiffin (meal) subscriptions, deductions, and remaining meals.

## Features

- **Plan Management**: Set total meals and start date.
- **Visual Calendar**: View consumed and skipped meals on a calendar.
- **Dynamic Deductions**: Automatically calculates consumed meals based on time of day (Lunch at 12 PM, Dinner at 8 PM).
- **Skip Meals**: Go back to any date and mark meals as skipped.
- **Shareable Image**: Generate a shareable image of your calendar and skipped meals history.
- **Dark Mode**: Sleek, glassmorphic dark theme.

## Tech Stack

- **Frontend**: React, Vite, Vanilla CSS
- **Backend**: Firebase (Firestore, Auth)
- **Libraries**: `html2canvas` (for sharing calendar)

## Getting Started

1. Clone the repo.
2. Install dependencies: `npm install`
3. Run the dev server: `npm run dev`
4. Open `http://localhost:5173` in your browser.
